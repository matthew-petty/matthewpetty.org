## Overview

Personal website for Matthew Petty with about, now, and blog pages. Static site using Hugo + PaperMod theme, hosted on GitHub Pages.

## Technical Stack

- **Generator:** Hugo
- **Theme:** PaperMod
- **Hosting:** GitHub Pages (with GitHub Action for builds)
- **Domain:** matthewpetty.org

## Site Structure

| Page  | Route   | Description                                          |
| ----- | ------- | ---------------------------------------------------- |
| About | `/`     | Brief bio, professional focus, personal story, photo |
| Now   | `/now`  | Current focus, following nownownow.com convention    |
| Blog  | `/blog` | Posts listed reverse-chronologically                 |

## Features

- Dark mode toggle (enabled from launch)
- RSS feed with full post content
- Tags/categories: not at launch, architecture should support adding later
